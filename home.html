🚀 EventFlow

Sistema web para gerenciamento de eventos, permitindo visualizar, filtrar e cadastrar eventos com localização no mapa.

📌 Sobre o projeto

O EventFlow é uma aplicação web para gerenciamento de eventos, permitindo visualizar, cadastrar, editar e excluir eventos, com suporte a localização no mapa e compartilhamento.

O projeto foi desenvolvido seguindo o padrão MVC (Model-View-Controller), garantindo organização, escalabilidade e separação de responsabilidades.

🎯 Objetivo

  • Estruturação com HTML
  • Estilização com CSS / Tailwind
  • Manipulação do DOM com JavaScript
  • Trabalho com JSON
  • Integração com APIs
  • Uso do padrão MVC
  • CRUD completo
  • Autenticação básica

🧱 Arquitetura (MVC)

  • Model: Gerencia dados (localStorage)
  • View: Interface com HTML + Tailwind
  • Controller: Lógica e manipulação do DOM

✨ Funcionalidades

  • Listagem de eventos (API + localStorage)
  • Cadastro de eventos
  • Edição de eventos
  • Exclusão de eventos
  • Busca por nome
  • Filtro por categoria
  • Compartilhamento com link
  • QR Code do evento

📍 Localização

  • Busca de endereço via CEP (ViaCEP)
  • Mapa interativo com Leaflet.js
  • Seleção de localização no mapa
  • Link direto para Google Maps

🖼️ Conteúdo do evento

  • Banner do evento (URL)
  • Documento explicativo
  • Descrição completa

🔐 Autenticação

  • Login básico
  • Proteção de rotas
  • Sessão com localStorage

🌐 Tecnologias

  • HTML5
  • CSS3 / TailwindCSS
  • JavaScript (ES6+)
  • LocalStorage
  • ViaCEP API
  • Leaflet.js

🚀 Melhorias futuras

  • Backend real (Laravel / Node)
  • Banco de dados
  • Upload real de imagens
  • Sistema de usuários completo
  • Deploy da aplicação
